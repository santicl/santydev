Detalles de este proyecto

Este proyecto fue creado con lenguajes como HTML, CSS, JavaScript. diseñado con metodología CRUD (Create, Read, Update, Delete).

Con este planificador de metas podrás saber cuanto deberás pagar por mes, además que te dará consejos de qué deberías hacer con tu ahorro.

Este proyecto cuenta con otra sección donde puedes lograr calcular el estado de resultados de tu negocio o empresa, ayudandote además para saber la rentabilidad y el ROI de tu empresa.

Esa sección te dará una idea clara deque saludable esta tu empresa.
